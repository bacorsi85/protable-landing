## NOTAS DE PRODUCCION

### Musica
- **Track principal:** Lounge/jazz suave instrumental. Estilo "cafe europeo". Sin letra.
- **Lado izquierdo:** La musica se siente mas lejana, filtrada, como si viniera de otro lado.
- **Lado derecho:** La musica es clara, presente, envolvente.
- **Crescendo final** en el cierre cuando aparece el logo.

### Efectos de sonido clave
| Momento | Efecto |
|---|---|
| Cronometro avanzando | Tick metalico suave, constante |
| Escaneo de QR | Sonido tech sutil (beep corto) |
| Agregar al carrito | Pop/ding positivo |
| Confirmar pedido | Whoosh + chime satisfactorio |
| Senal viajando a cocina | Whoosh electrico con trail |
| Kanban moviendo columna | Click + slide suave |
| Copas brindando | Clink cristalino |
| Pago confirmado | Ka-ching moderno |
| Freeze frame izquierda | Corte abrupto de audio |

### Paleta de la interfaz ProTable (para las tomas de UI)
| Elemento | Color |
|---|---|
| Primario / Acentos | Violeta #7C3AED |
| Fondo app | Blanco #FFFFFF |
| Texto principal | Negro #1A1A1A |
| Texto secundario | Gris #6B7280 |
| Exito/Confirmacion | Verde #10B981 |
| Carrito/Badge | Violeta con numero blanco |
| Botones principales | Violeta con texto blanco, bordes redondeados |

### Indicaciones de grabacion
1. **Grabar ambos lados por separado** con la misma pareja, mismo restaurante, mismo dia.
2. **La iluminacion cambia entre lados:** izquierda mas fria (4000K), derecha mas calida (3200K).
3. **El cronometro es un overlay fijo** en la parte superior central, visible durante todo el video.
4. **La linea divisoria** del split screen es una linea fina violeta (2px) con leve glow.
5. **Las pantallas de UI** se graban aparte (screen recording del prototipo) y se componen en post.
6. **La transicion de la senal digital** (pedido viajando a cocina) es full motion graphics, se hace en After Effects o similar.

### Versiones a entregar
- **Full (90-120s):** Para landing page y YouTube
- **Corta (30s):** Para Instagram Reels / TikTok (solo escenas 3, 4 y cierre)
- **Ultra-corta (15s):** Para stories / ads (solo el split de entrega del vino vs. seguir esperando + CTA)
